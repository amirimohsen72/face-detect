# face-detect
detect face from image

Getting started
this project use venv:

setup venv :
````
python -m venv venv
```

run venv :
```
venv\scripts\activate
```

install packages :
```
pip install -r requirements.txt
```
or use 
[X] pip install matplotlib
[X] pip install opencv-python

you need image and model.xml in this project.
for run use this:
```
python main.py
```
